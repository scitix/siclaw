# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Siclaw, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please use [GitHub Security Advisories](https://github.com/scitix/siclaw/security/advisories/new) to report the vulnerability privately.

Alternatively, you can email the maintainers directly. We will acknowledge your report within 48 hours and work with you to understand and address the issue.

## Scope

This policy applies to the Siclaw codebase hosted at [github.com/scitix/siclaw](https://github.com/scitix/siclaw).

|---------|-----------|
